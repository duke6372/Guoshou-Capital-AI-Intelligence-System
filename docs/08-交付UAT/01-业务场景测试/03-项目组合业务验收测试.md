# 项目组合业务验收测试

## 模块概述

项目组合模块提供组合内项目的统一视图，支持项目管理、对标分析和风险监控。

## 业务价值验证点

1. **项目全景视图**: 全面掌握组合内所有项目
2. **快速筛选定位**: 按城市、业态等快速找到目标项目
3. **同城同业态对标**: 与同类项目对比分析
4. **项目详情穿透**: 一键查看项目所有信息

## 测试用例

### UAT-PF-001: 查看项目列表

**场景描述**:
我需要查看组合内所有项目的基本信息。

**前置条件**:
- 用户已登录
- 有项目数据

**测试步骤**:
1. 点击左侧菜单"项目组合"
2. 观察项目列表

**预期结果**:
| 列 | 内容 |
|----|------|
| 风险 | 红/黄/正常标签 |
| 业态 | 如商业/物流/园区 |
| 项目 | 项目名称 |
| 关键指标 | 出租率/NOI等 |
| 操作 | 查看详情入口 |

**优先级**: P0
