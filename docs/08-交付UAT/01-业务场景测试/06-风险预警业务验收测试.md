# 风险预警业务验收测试

## 模块概述

风险预警模块提供预警规则配置、预警触发、推送通知和处置闭环功能，帮助风控人员及时发现和处理风险。

## 业务价值验证点

1. **预警及时准确**: 触发条件合理，不漏报不误报
2. **处置流程清晰**: 确认→指派→处置→关闭，规范闭环
3. **操作完整留痕**: 所有操作可追溯审计
4. **推送通道畅通**: 预警及时送达责任人

## 测试用例

### UAT-RG-001: 查看预警列表

**场景描述**:
我需要查看当前所有预警。

**前置条件**:
- 用户已登录
- 有预警数据

**测试步骤**:
1. 点击左侧菜单"风险预警"
2. 观察预警列表

**预期结果**:
| 列 | 内容 |
|----|------|
| 级别 | 红/黄/提示标签 |
| 类型 | 出租率/租约到期/能耗等 |
| 项目 | 项目名称 |
| 当前值 | 实际数值 |
| 阈值 | 触发阈值 |
| 状态 | 待确认/已确认/已关闭 |
| 创建时间 | 预警生成时间 |

**优先级**: P0

### UAT-RG-003: 确认预警

**场景描述**:
我看到预警后需要确认这是真实风险。

**前置条件**:
- 存在待确认预警

**测试步骤**:
1. 选择目标预警
2. 点击"确认"
3. 填写确认原因（如"正常波动"或"需关注"）

**预期结果**:

- 确认操作被记录
- 后续可指派处置

**验证方式**:
检查预警状态是否正确更新

**优先级**: P0

## 预警规则说明

| 规则 | 默认阈值 | 持续时间 | 升级策略 |
|------|----------|----------|----------|
| 出租率预警 | <90% | 连续7天 | 黄→红 |
| 租约到期预警 | >30%即将到期 | - | 黄 |
| 能耗异常预警 | 偏离>12% | 连续3天 | 蓝→黄 |
| 客流异常预警 | 同比<-8% | 连续7天 | 黄 |

## 预警级别定义

| 级别 | 含义 | 处理要求 | 推送方式 |
|------|------|----------|----------|
| 红色 | 紧急需立即处理 | 2小时内响应 | 短信+系统+邮件 |
| 黄色 | 重要需关注 | 24小时内响应 | 系统+邮件 |
| 蓝色/提示 | 一般提醒 | 下一工作日 | 系统 |
