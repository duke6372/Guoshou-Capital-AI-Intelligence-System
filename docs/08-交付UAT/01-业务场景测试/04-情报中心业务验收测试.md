# 情报中心业务验收测试

## 模块概述

情报中心提供竞争情报、政策动向、行业动态等多类情报的采集、管理和推送功能。

## 业务价值验证点

1. **情报一站式获取**: 集中看到所有关注的情报
2. **智能分类筛选**: 快速找到需要的情报
3. **订阅自动推送**: 不需要手动查询，定期收到推送
4. **影响评估辅助**: AI辅助判断情报影响

## 测试用例

### UAT-INT-001: 查看情报列表

**场景描述**:
我需要浏览所有最新情报。

**前置条件**:
- 用户已登录
- 有情报数据

**测试步骤**:
1. 点击左侧菜单"情报中心"
2. 观察情报列表

**预期结果**:
每条情报显示：
- 类型标签（竞争/政策/运营/行业）
- L级别标签（L1-L5）
- 置信度标签（高/中/低）
- 标题
- 摘要
- 来源
- 时间

**优先级**: P0

## 情报分类定义

| 类型 | 说明 | 示例 |
|------|------|------|
| 竞争 | 竞争对手动态 | 竞品新项目、投拓动向 |
| 政策 | 政策法规 | 城市更新、REITs新规 |
| 运营 | 运营相关 | 行业数据、市场动态 |
| 行业 | 行业发展 | 技术创新、趋势报告 |
| 其他 | 综合信息 | 合作机会、人才流动 |

## 情报L级别定义

| 级别 | 含义 | 说明 |
|------|------|------|
| L1 | 宏观影响 | 影响整个行业 |
| L2 | 重大影响 | 影响公司战略 |
| L3 | 中等影响 | 影响部分业务 |
| L4 | 一般影响 | 影响具体项目 |
| L5 | 轻微影响 | 信息参考 |
