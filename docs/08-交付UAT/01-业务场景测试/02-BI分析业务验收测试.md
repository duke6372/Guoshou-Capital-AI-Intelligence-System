# BI分析业务验收测试

## 模块概述

经营分析（BI）模块提供组合级别的经营指标分析，帮助用户深入理解业务数据，支持决策。

## 业务价值验证点

1. **指标趋势洞察**: 看清指标的长期变化趋势
2. **维度拆解分析**: 从城市、业态、项目等维度拆解指标
3. **口径透明可追溯**: 数据计算规则清晰可查
4. **数据导出留痕**: 支持导出且有完整审计

## 测试用例

### UAT-BI-001: 查看BI首页KPI卡片

**场景描述**:
作为分析人员，我需要快速看到核心BI指标。

**前置条件**:
- 用户已登录
- BI模块已配置

**测试步骤**:
1. 进入系统
2. 点击左侧菜单"经营分析"
3. 观察顶部KPI卡片区域

**预期结果**:
| 指标 | 显示 | 说明 |
|------|------|------|
| 出租率 | 92.4% | 组合级别出租率 |
| 租金收入 | ¥4.36亿 | 本月收入 |
| NOI | ¥2.84亿 | 本月净运营收入 |
| 数据质量 | 86分 | 数据完整性/准确性评分 |

**验证方式**:
与数据源系统核对

**优先级**: P0

## 数据口径确认清单

| 指标 | 口径说明 | 业务确认 |
|------|----------|----------|
| 出租率 | (已出租面积/总可出租面积)×100%，排除装修期 | □ |
| 租金收入 | 当月实际到账租金 | □ |
| NOI | 运营收入-运营成本（不含折旧摊销） | □ |
| 数据质量 | 完整性×30%+准确性×40%+时效性×30% | □ |
