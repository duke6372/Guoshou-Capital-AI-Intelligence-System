# Copilot业务验收测试

## 模块概述

Copilot提供智能问答、报告生成、引用溯源等功能，是系统的AI智能助手。

## 业务价值验证点

1. **自然语言交互**: 用自然语言提问，快速获取答案
2. **答案可溯源**: 回答附带引用来源，可追溯验证
3. **一键生成报告**: 快速生成各类报告
4. **全模块覆盖**: 覆盖经营/情报/知识/风险各场景

## 测试用例

### UAT-CP-001: 使用快捷问题

**场景描述**:
我想快速获取NOI变化的原因分析。

**前置条件**:
- Copilot功能正常
- 系统有NOI数据

**测试步骤**:
1. 在工作台找到"Copilot快捷提问"面板
2. 点击"解释本月NOI变化原因（按项目/业态拆解）"
3. 等待AI生成回答

**预期结果**:
- 显示加载状态
- AI返回归因分析
- 回答包含Top贡献项目/业态
- 展示数据引用来源

**验证方式**:
业务人员确认分析结果与业务理解一致

**优先级**: P0

### UAT-CP-002: 自由问答

**场景描述**:

### UAT-CP-003: 查看回答中的引用

**场景描述**:
我想验证AI回答的依据。

**前置条件**:
- Copilot返回了回答

**测试步骤**:
1. 观察AI回答中的引用标注（如[1][2]）
2. 点击引用标注
3. 查看引用来源详情

**预期结果**:
- 回答中有引用标注
- 底部或侧边显示引用清单
- 点击可查看引用原文
- 引用与回答相关

**验证方式**:
核对引用来源，确认回答准确性

**优先级**: P0

## Copilot使用场景

| 场景 | 快捷问题示例 | 功能价值 |
|------|--------------|----------|
| 经营分析 | "解释本月NOI变化" | 快速归因 |
| 情报研究 | "分析竞品动态影响" | 情报解读 |
| 知识问答 | "租赁合同有哪些风险点" | 知识检索 |
| 风险预警 | "哪些项目需要关注" | 风险识别 |
| 报告生成 | "生成高管日报" | 一键报告 |

## AI回答质量评估

业务人员评估AI回答时，关注以下方面：

| 维度 | 评估标准 |
|------|----------|
| 准确性 | 回答中的事实是否正确 |
| 完整性 | 是否完整回答了问题 |
| 相关性 | 回答是否切题 |
| 可解释性 | 引用来源是否充分 |
| 可操作性 | 建议是否可执行 |
