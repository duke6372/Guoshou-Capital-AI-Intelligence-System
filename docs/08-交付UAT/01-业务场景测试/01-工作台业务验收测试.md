# 工作台业务验收测试

## 模块概述

工作台是系统的首页，为用户提供核心经营指标的快速概览，帮助高管及时了解公司运营状况。

## 业务价值验证点

1. **一眼看清全局**: 核心指标一目了然
2. **快速发现异常**: 异常情况突出显示
3. **快速处理待办**: 待办事项清晰可操作
4. **智能辅助决策**: Copilot快速获取洞察

## 测试用例

### UAT-WB-001: 查看核心KPI指标

**场景描述**:
作为高管，我登录系统后希望第一眼看到公司的核心经营指标，以便快速了解当前运营状况。

**前置条件**:
- 用户已获得系统访问账号
- 系统已完成首次数据同步
- 当日数据已更新

**测试步骤**:
1. 打开浏览器，访问系统登录页
2. 输入账号密码登录
3. 系统自动进入工作台页面
4. 观察页面顶部4个KPI卡片

**预期结果**:
| 指标卡片 | 应显示内容 | 业务含义 |
|----------|------------|----------|
| 在管规模 | 约¥1,700亿 | 公司签约管理的不动产规模 |
| 组合出租率 | 约92.4% | 整体出租率水平 |
| NOI | 约¥2.84亿 | 本月净运营收入 |
| 风险预警 | 10条未关闭 | 当前需要关注的预警数量 |

**验证方式**:
与财务部门提供的同期数据对比，确认数据准确（允许误差<1%）

**优先级**: P0

### UAT-WB-002: 查看指标变化趋势

**场景描述**:
我需要了解核心指标的变化趋势，判断业务走势。

**前置条件**:
- 已登录系统进入工作台
- 系统有历史数据积累

**测试步骤**:
1. 在工作台找到"经营趋势"图表
2. 观察出租率趋势曲线
3. 查看图表下方显示的时间范围

**预期结果**:
- 趋势图显示过去12个月的出租率变化
- 曲线平滑，能看到整体趋势
- 有数据点标注，可看到具体数值

**验证方式**:
观察图表是否清晰易读，趋势是否符合预期

**优先级**: P0

### UAT-WB-003: 识别并处理异常预警

**场景描述**:
我需要及时发现和处理异常预警，确保业务风险可控。

**前置条件**:

**测试步骤**:
1. 在工作台查看风险预警卡片
2. 记录红色和黄色预警数量

4. 进入预警列表查看详情

**预期结果**:
- 卡片显示红/黄色预警数量
- 点击后进入预警详情页
- 可看到预警的详细信息（项目、指标、阈值）

**验证方式**:
检查预警信息与实际情况是否一致

**优先级**: P0
