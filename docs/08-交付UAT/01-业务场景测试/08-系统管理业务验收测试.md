# 系统管理业务验收测试

## 模块概述

系统管理模块提供组织权限、数据接入、指标口径和审计等系统级配置功能。

## 业务价值验证点

1. **权限安全可控**: 最小权限原则，数据不泄露
2. **配置灵活调整**: 系统参数可调整适配业务
3. **操作全程审计**: 所有敏感操作留痕可查
4. **数据质量透明**: 数据接入状态可见

## 测试用例（系统管理员）

### UAT-ADM-001: 查看用户列表

**场景描述**:
管理员需要查看系统所有用户。

**前置条件**:
- 管理员已登录

**测试步骤**:
1. 点击左侧菜单"管理后台"
2. 找到"用户管理"或"组织架构"
3. 观察用户列表

**预期结果**:
| 列 | 内容 |
|----|------|
| 用户名 | 姓名 |
| 邮箱 | 登录账号 |
| 角色 | 分析师/风控/管理员等 |
| 状态 | 启用/禁用 |
| 所属部门 | 组织归属 |

**优先级**: P0

### UAT-ADM-002: 创建新用户

**场景描述**:

**前置条件**:
- 管理员权限

**测试步骤**:
1. 点击"新建用户"
2. 填写用户信息（姓名、邮箱、角色、部门）

**预期结果**:

- 用户可正常登录

**优先级**: P0

### UAT-ADM-005: 验证权限生效

**场景描述**:
我需要确认权限配置生效了。

**前置条件**:
- 已完成权限配置

**测试步骤**:
1. 用该用户账号登录
2. 尝试访问授权项目（应成功）

**预期结果**:

- 未授权项目不可见或返回403

## 角色权限说明

| 角色 | 权限范围 |
|------|----------|
| 管理员 | 全部功能、用户管理、系统配置 |
| 风控管理员 | 预警规则、推送配置、审计查看 |
| 数据Owner | 指标口径、数据接入管理 |
| 分析师 | 数据查看、分析、导出（受权限控制） |
| 普通用户 | 查看有权限的数据、使用Copilot |

## 审计操作范围

系统记录以下操作的完整日志：

| 操作类型 | 说明 |
|----------|------|
| 登录 | 用户登录/登出 |
| 数据查看 | 敏感数据访问 |
| 数据导出 | 数据导出操作 |
| 配置变更 | 系统配置修改 |
| 权限变更 | 用户权限修改 |
| 预警处置 | 预警确认/指派/关闭 |
