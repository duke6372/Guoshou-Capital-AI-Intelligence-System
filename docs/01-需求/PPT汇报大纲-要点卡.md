# PPT汇报大纲（要点卡）

## 这份材料在解决什么问题（目的）
- 为管理层汇报准备“可行性调研与实施方案”的PPT结构与口径，帮助在30–40分钟内讲清楚为什么做、做什么、怎么做、要投入什么、能带来什么价值。

## 覆盖的业务域/对象（范围）
- 管理对象：国寿资本在管不动产投资组合（含境内/境外、多业态）、以及与经营分析/投后管理相关的数据、情报、知识与风险。
- 目标受众：国寿资本/中国人寿投资中高管理层（汇报对象明确）。

- “为什么现在”：规模与复杂度上升 + 三大痛点（数据分散/情报滞后/知识流失）+ 竞品压力 + 大模型窗口成熟。[PPT汇报大纲.pdf.md:L38-L45](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/PPT%E6%B1%87%E6%8A%A5%E5%A4%A7%E7%BA%B2.pdf.md#L38-L45)
- 总体目标与四个具体目标：数据自动化、情报主动化、知识沉淀化、风险前置化（含效率指标）。[PPT汇报大纲.pdf.md:L50-L56](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/PPT%E6%B1%87%E6%8A%A5%E5%A4%A7%E7%BA%B2.pdf.md#L50-L56)
- 六大模块与整体分层架构：DataHub / IntelEngine / KnowledgeBase / BIDashboard / RiskGuard / Copilot。[PPT汇报大纲.pdf.md:L76-L81](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/PPT%E6%B1%87%E6%8A%A5%E5%A4%A7%E7%BA%B2.pdf.md#L76-L81)
