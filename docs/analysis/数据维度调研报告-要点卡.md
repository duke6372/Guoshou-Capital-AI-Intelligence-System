# 数据维度调研报告（要点卡）

## 这份材料在解决什么问题（目的）
- 梳理AI智能体情报系统可收集的数据资产目录（内部经营数据+外部行业数据），为DataHub数据中枢的接入范围、采集频率、数据治理提供依据。

## 覆盖的业务域/对象（范围）
- 内部：投资组合层、项目运营、合同法务、工程建设、人力与决策流程等数据维度。[数据维度调研报告.pdf.md:L24-L53](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L24-L53)
- 外部：宏观市场、地产细分市场、竞品动态、政策监管、城市规划等数据维度。[数据维度调研报告.pdf.md:L53-L83](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L53-L83)

- 数据分两大类：内部经营数据与外部行业数据，是平台上层AI分析的原材料。[数据维度调研报告.pdf.md:L20-L23](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L20-L23)
- 典型内部指标：IRR/MOIC/DPI/TVPI、出租率/租金/NOI/EBITDA、租约条款与续约历史、投审会记录与决策周期等。[数据维度调研报告.pdf.md:L27-L53](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L27-L53)
- 典型外部指标与数据源：GDP/利率/REITs、写字楼租金指数/空置率、商业客流与坪效、物流仓储高标仓、竞品项目/基金/人事动态、政策与规划等。[数据维度调研报告.pdf.md:L56-L83](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L56-L83)
- 数据治理建议：MDM主数据、血缘追踪、质量评分（完整性/准确性/时效性）、安全分级。[数据维度调研报告.pdf.md:L90-L93](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L90-L93)

- 平台需支持多频率采集：季度/月度/实时（尤其是客流、IoT能耗、竞品动态、政策订阅）。[数据维度调研报告.pdf.md:L35-L39](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L35-L39)
- 需要统一主数据：项目、资产、客户/租户、竞品主体、合同等标识体系，支撑跨系统对齐。[数据维度调研报告.pdf.md:L90-L91](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L90-L91)

- 外部依赖：统计局、Wind、交易所、代理行、赢商、中指院、企查查/天眼查、监管与政府官网等。[数据维度调研报告.pdf.md:L56-L82](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E6%95%B0%E6%8D%AE%E7%BB%B4%E5%BA%A6%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L56-L82)

- 内部系统数据可用性：字段粒度、历史数据回溯范围、刷新频率、是否可API直连。
- “指标口径字典”优先级：高管驾驶舱必须统一的Top 20指标是什么，定义与计算规则由谁维护。
