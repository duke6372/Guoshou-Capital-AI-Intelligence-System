# 竞争对手调研报告（要点卡）

## 这份材料在解决什么问题（目的）
- 梳理保险系不动产投资的主要竞争者（平安/泰康/友邦/新华）的投资策略与数字化成熟度，形成对标框架，反推国寿资本AI智能体情报系统的差异化定位与必做能力。

## 覆盖的业务域/对象（范围）
- 对象：竞争对手平台（不动产投资平台/投后系统/数据中台）、竞争对手资产布局（商业/物流/公寓/产业园/医养等）、数字化能力（数据管理、投后系统、AI应用方向）。[竞争对手调研报告.pdf.md:L20-L38](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L20-L38)

## 关键结论/事实（可追溯）
- 平安不动产：已建数据中台、在投后管理系统（REPMS）投入大，数字化成熟度高，是核心对标对象。[竞争对手调研报告.pdf.md:L28-L37](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L28-L37)
- 竞争格局总结：平安领先、泰康中高、新华中、友邦低；国寿需加快节奏并形成差异化优势表达。[竞争对手调研报告.pdf.md:L79-L85](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L79-L85)
- 对系统的启示：竞品动态监控、对标数据体系、差异化定位、机会识别。[竞争对手调研报告.pdf.md:L86-L92](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L86-L92)

## 明确的需求/能力点（用户故事/功能点）
- 竞品动态追踪：围绕“项目获取/融资/人事/战略/处置”等主题的监控与归档，支持早期信号发现与检索复盘。[竞争对手调研报告.pdf.md:L86-L88](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L86-L88)
- 对标数据库：围绕出租率、租金、收益率等核心指标实现横向对比，支持“同城/同业态/同阶段资产”对标口径。[竞争对手调研报告.pdf.md:L88-L89](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L88-L89)
- 机会识别：基于竞品布局发现市场空白点（城市/业态/策略）。[竞争对手调研报告.pdf.md:L91-L91](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E7%AB%9E%E4%BA%89%E5%AF%B9%E6%89%8B%E8%B0%83%E7%A0%94%E6%8A%A5%E5%91%8A.pdf.md#L91-L91)

## 数据与系统边界（数据源、上下游、依赖）
- 数据来源：公开新闻、行业报告、工商数据平台、社交媒体与监管公示等；需要与“情报体系（L3）”的主题与置信度机制打通。

## 未决问题（需要向业务/技术确认）
- “重点对标清单”：需要持续跟踪的竞品主体与子品牌、重点城市/业态范围、以及对标指标的口径与可获取性。
- “差异化叙事”：国寿资本要强调的独特价值（保险主业协同、长期资金优势、合资治理经验等）在系统里如何被产品化呈现（页面/报告/指标）。
