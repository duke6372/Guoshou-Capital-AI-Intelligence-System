# 国寿资本AI智能体情报系统：背景综合（跨材料）

## 1. 一句话定位
- 面向国寿资本/中国人寿投资中高管理层的“AI驱动情报与决策支持平台”，以数据底座为基础，将经营分析（BI）、情报（竞品/政策/市场）、知识（合同/制度/案例）与风险预警整合在一个可追溯、可审计的体系内。

## 2. 为什么现在（业务动因）
- 管理规模与复杂度上升：22只在管基金、逾1700亿元签约规模，组合覆盖境内/境外、多业态。[咨询建议书.pdf.md:L36-L41](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E5%9B%BD%E5%AF%BF%E8%B5%84%E6%9C%ACAI%E6%99%BA%E8%83%BD%E4%BD%93%E6%83%85%E6%8A%A5%E7%B3%BB%E7%BB%9F%E5%92%A8%E8%AF%A2%E5%BB%BA%E8%AE%AE%E4%B9%A6.pdf.md#L36-L41)
- 三大痛点集中爆发：数据分散、情报滞后、知识流失。[汇报大纲.pdf.md:L40-L44](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/PPT%E6%B1%87%E6%8A%A5%E5%A4%A7%E7%BA%B2.pdf.md#L40-L44)
- 竞品压力与技术窗口：头部对手（平安不动产）数字化成熟度高；国产大模型成熟，AI应用门槛降低。[汇报材料.pptx.md:L33-L41](file:///Users/mic/Documents/trae_projects/Guoshou/docs/source/%E5%9B%BD%E5%AF%BF%E8%B5%84%E6%9C%ACAI%E6%99%BA%E8%83%BD%E4%BD%93%E6%83%85%E6%8A%A5%E7%B3%BB%E7%BB%9F%E6%B1%87%E6%8A%A5%E6%9D%90%E6%96%99.pptx.md#L33-L41)
