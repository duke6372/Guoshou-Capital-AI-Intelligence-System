# 端到端场景测试

本文档定义跨模块的端到端集成测试，覆盖核心用户旅程和关键业务场景。

## 高管日报旅程 (D1)

### E2E-D1-001: 看总览 → 看异常 → 看穿透 → 导出

**测试目标**: 验证高管5分钟内完成核心工作

**前置条件**:
- 测试用户已登录
- 数据已同步到最新
- 用户有导出权限

**测试步骤**:
1. 进入工作台页面
2. 观察KPI卡片数据（4项核心指标）
3. 查看异常摘要卡片
4. 点击异常卡片进入详情
5. 返回工作台，点击Top异常项目
6. 跳转到项目详情
7. 返回工作台，点击"生成报告"
8. 生成高管日报
9. 导出报告

**预期结果**:
| 步骤 | 验证点 | 预期时间 |
|------|--------|----------|
| 1 | KPI卡片完整显示 | <2s |
| 2 | 异常摘要正确 | 立即 |
| 3 | 详情跳转正确 | <1s |
| 5 | 项目详情正确 | <2s |
| 7 | 报告生成成功 | <10s |
| 9 | 导出文件正确 | <3s |

**总耗时**: <30秒（不含等待）

**优先级**: P0
**测试类型**: E2E测试
