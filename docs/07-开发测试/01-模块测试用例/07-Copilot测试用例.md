# Copilot测试用例

## 模块概述

Copilot提供智能问答、报告生成、引用溯源和审计留痕功能。

**功能清单**:
- 智能问答（经营/情报/知识/风险）
- 一键报告生成
- 引用来源展示
- 审计留痕
- 回答反馈

**测试数据**:

- 负面样本: 50条其他模块问题

**评估指标**:
- 准确率 ≥ 90%
- 精确率 ≥ 85%
- 召回率 ≥ 88%

**测试方法**:
```python
# 测试代码示例
def test_intent_classification():
    test_cases = load_test_data("intent_classification_test.json")
    correct = 0
    for case in test_cases:
        predicted = classify_intent(case["query"])
        if predicted == case["expected_intent"]:
            correct += 1
    accuracy = correct / len(test_cases)
    assert accuracy >= 0.90
```

**优先级**: P0
**测试类型**: AI评估
