# 项目管理计划

## 1. 概述

本文档涵盖国寿资本AI智能体情报系统的项目管理计划，包括生命周期模型选择、阶段划分、交付物定义和资源安排。

## 2. 开发生命周期

采用**混合敏捷**模式：

- **整体规划**：里程碑驱动的瀑布式阶段规划（Phase 0 ~ Phase 5）
- **迭代执行**：每个Phase内采用Scrum敏捷开发，2周一迭代
- **验收节奏**：每Phase结束后统一演示与验收

## 3. 阶段与交付物

| 阶段 | 核心任务 | 关键交付物 |
|------|----------|------------|
| Phase 0 MVP | 最小可用试点 | 工作台、驾驶舱(D1)、项目组合(D2)、项目详情(D3)、基础预警 |
| Phase 1 数据底座 | 数据治理与整合 | 指标口径字典、数据血缘、ETL管道、质量评分机制 |
| Phase 2 BI能力 | 完整分析能力 | 完整BI看板、对标分析、经营报告导出、移动端适配 |
| Phase 3 知识库 | 知识沉淀与复用 | 知识库管理、RAG问答、合同审查、条款比对 |
| Phase 4 情报引擎 | 情报闭环 | 情报采集、置信度评估、影响分析、报告自动生成 |
| Phase 5 Copilot | AI助手 | NLP查询、跨模块意图识别、报告协作、订阅推送 |

## 4. 迭代节奏

| 活动 | 频率 | 说明 |
|------|------|------|
| 每日站会 | 每日15min | 同步进度、识别阻塞 |
| 冲刺规划 | 每2周 | 确定冲刺待办列表 |
| 冲刺演示 | 每2周 | 向干系人演示增量功能 |
| 回顾会议 | 每2周 | 团队内部改进讨论 |
| 阶段评审 | 每Phase结束 | 正式评审与验收签字 |
