## 2. 项目背景

国寿资本管理22只在管基金、逾1700亿元签约规模，组合覆盖境内/境外、多业态。当前面临三大核心痛点：

- **数据分散**：多系统分散、口径不一，报表制作周期3-5天
- **情报滞后**：竞品与政策缺乏系统追踪，信息碎片化
- **知识流失**：合同知识与决策经验未沉淀，依赖个人经验

本项目旨在建设一套AI驱动的情报与决策支持平台，将经营分析（BI）、情报研判、知识复用与风险预警整合在统一可追溯、可审计的体系内。

## 3. 项目目标

### 3.1 北极星目标
让管理层与一线团队从"找数据/找材料/找人问经验"转变为"随时可查、可比、可追溯、可预警、可复用"，提升经营分析效率与决策质量。

### 3.2 可量化目标

| 指标 | 现状 | 目标 |
|------|------|------|
| 报表生成效率 | 3-5天 | 即时/分钟级 |
| 竞品与政策响应 | 事后被动知晓 | 周级预警/24h解读 |
| 合同审查效率 | 2-3天 | 2-3小时（AI+人工复核）|
| 风险发现 | 事后发现 | 事前预警（30天以上）|

## 4. 项目范围

### 4.1 包含范围（六大核心模块）

| 模块 | 代号 | 说明 |
|------|------|------|
| 数据底座 | M1 DataHub | 多源数据采集与整合，统一口径、质量与血缘 |
| 情报引擎 | M2 IntelEngine | 竞品/政策情报采集、分类、置信度评估与影响分析 |
| 知识库 | M3 KnowledgeBase | 合同/制度/案例等知识体系 + RAG问答 |
| BI看板 | M4 BIDashboard | 驾驶舱、组合视图、项目详情、对标分析 |
| 风险预警 | M5 RiskGuard | 量化阈值预警与分级推送机制 |
| AI助手 | M6 Copilot | 自然语言驱动的查询、研判与报告协作 |

### 4.2 不包含范围

- 核心财务系统的重建或替换
- 线下纸质档案的全面数字化（由客户方数据团队负责）
- 第三方数据源的采购谈判（由客户方商务团队负责）
